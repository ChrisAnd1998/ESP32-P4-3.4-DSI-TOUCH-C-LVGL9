# ESP32-P4-3.4-DSI-TOUCH-C-LVGL9
A simple Arduino IDE project to create a LVGL 9 project using a ESP32-P4 Module DEV KIT and Waveshare 3.4inch DSI Round Capacitive Touch Display, 800×800 Resolution, IPS Display Panel, 10-Point Touch (3.4-DSI-TOUCH-C)
